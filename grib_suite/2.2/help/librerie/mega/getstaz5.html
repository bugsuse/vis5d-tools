	SUBROUTINE GETSTAZ5(ALAMX,ALAMN,ALOMX,ALOMN,HMX,HMN,
	1	   IWMO1,IWMO2,
	1	IWMO,ANOME,HHSLM,HHPOZ,ALAT,ALON,IER)

	
	Serve ad estrarre dall'archivio meteodata le stazioni ed i
	relativi parametri anagrafici letti dall' archivio meteodata
	contenute nei limiti definiti lat. lon. alt.

	Input	:	ALATMX	= R*4	Latitudine max finestra
			ALATMN	= R*4	Latitudine min finestra
			ALONMX	= R*4	Longitudine max finestra	
			ALONMN	= R*4	Longitudine min finestra
			HMX	= R*4	Altezza max finestra
			HMN	= R*4	Altezza min finestra
			IWMO1	= I*4	Codice W.M.O. stazione min finestra
			IWMO2	= I*4	Codice W.M.O. stazione max finestra
			IWMO    = I*4	inizializzato ad un valore inferiore
					ad IWMO1 per il primo accesso

	Output	:	IWMO	= I*4	Codice W.M.O. stazione
			ANOME   = C*24	Nome della stazione
			HHSLM	= R*4	Altezza stazione S.L.M.(m.)
			HHPOZ	= R*4   Altezza del pozzetto (m.)
			ALAT	= R*4	Latitudine ( Gradi,primi)
			ALON	= R*4	Longitudine (Gradi,primi)
			IER	= I*4	0=O.K.; >0=errore fatale;
					<0=segnalazione errore triviale

	Chiamate 		: nessuna
	COMMON	Utilizzati	: /CONTRO/	CONTRO(12) CONTIENE IL NUMERO DI
						UNITA` DA UTILIZZARE
				  /ESTRASTAZ/	SEZ2 CON TUTTI I DATI ANAGRAFICI
						DELLA STAZIONE CORRETTI SOLO SE
						SONO STATI TROVATI E LETTI

	IER=-2	record non trovato
	IER=0	tutto O.K.
	IER=1	errore apertura file archivio stazioni
	IER=2	fine file
	IER=3	altri errori su lettura record

Torna all'indice

Torna alla voce superiore
