	SUBROUTINE TABSPLIMSL (DATI,N,N1,N2,MX,MN,MD,IN,IER)

	Questa ruotine valuta il valore massimo, minimo, medio di un insieme
	di dati utilizzando un interpolazione "spline" dei dati.
	Fa un controllo sui dati in maniera temporale controllando
	che il dato considerato non sia un massimo o un minimo e che
	contemporaneamente non differisca dai valori intorno piu` di
	0.1 gradi C per minuto.
	L'indice del vettore dati e` l'asse dei tempi che vengono
	ricalcolati dalla funzione IMINUTI.
	
	DATI(N)	: 	I*2	VETTORE CONTENENTE TUTTI I DATI
	N1	:	I*4	INDICE DELL' ELEMENTO DA CUI SI INIZIA
				L'INTERPOLAZIONE
	N2	:	I*4	INDICE DELL' ELEMENTO CON CUI SI FINISCE
				L'INTERPOLAZIONE
	MX	:	REAL	VALORE MASSIMO STIMATO
	MN	:	REAL	VALORE MINIMO STIMATO
	MD	:	REAL	VALORE MEDIO STIMATO
	IN	:	I*4	NUMERI DI DATI BUONI TROVATI
	IER	:	I*4	INDICATORE DI ERRORE

	IER =  0    : 	TUTTO O.K.
	IER =  1    :   L'indice N2 e' > di N o e' > di 100	
	IER =  2    :	Nessun dato buono nell' intervallo N1-N2 oppure
			fra N1 e N2 esiste un buco fra i dati superiore a 2 ore
	IER =  3    :   Ci sono piu` di tre dati considerati errati

Torna all'indice
