	SUBROUTINE  REGLH_E(AIN,AOUT,IM,JM,IMHV,JMHV)

	INPUT:
	AIN(IM,JM)	REAL	MATRICE ENTRANTE
	IM		INTEGER	
	JM		INTEGER	
	IMHV		INTEGER	
	JMHV		INTEGER	

	OUTPUT:
	AOUT(imhv,jmhv)

=======================================================================

	Spacchetta la matrice entrante nell'indice j: ricordiamo
	che sulla griglia E-Arakawa staggered:

      	         h   u,v   h   u,v   h   ...  u,v   h
	j=jmt
	        u,v   h   u,v   h   u,v  ...   h   u,v
	        ...  ...  ...  ...  ...  ...  ...  ...
	j
	        ...  ...  ...  ...  ...  ...  ...  ...
	        u,v   h   u,v   h   u,v  ...   h   u,v
	j=2
                h   u,v   h   u,v   h   ...  u,v   h
	        u,v   h   u,v   h   u,v  ...   h   u,v
	j=1
                h   u,v   h   u,v   h   ...  u,v   h


	       i=1   i=2  i=3  i=4  i=5   i       i=imt



	La matrice AIN e' compattata nell'indice j; la matrice AOUT
	e' scompattata, ed ha infatti una dimensione j doppia di AIN.

	N.B.: AOUT(i,j) ha un valore di h per ogni (i,j).

-----------------------------------------------------------------------

Torna all'indice

Torna alla voce superiore
