 SUBROUTINE SET_CLUSTER (K,LAT,LON,HSTAZ,IMESE,ICLUSTER,IER)

	Questa routine definisce, dato un vettore di stazioni, il cluster di
	appartenenza di ogni stazione. I cluster sono stati definiti sulla base
	della climatologia indentificata da Nanni.

	Vengono definiti cluster diversi a seconda del mese relativo.

===============================================================================
	Sempre valido

	CLUSTER = 0 		Fuori dai limiti regionali
				(lat. min.= 43.4, lat. max.= 45.1,
				 lon. min.=  9.1, lon. max.= 12.5,
				 definiti in centesimali)
===============================================================================
	OTTOBRE-APRILE (Inverno)

	CLUSTER = 0 		Stazione di montagna HSTAZ > 500. metri
	CLUSTER = 1		Stazione di pianura
===============================================================================
	AGOSTO-SETTEMBRE

	CLUSTER = 0		Se Lat.< 44.7 e Lon < 10.5
	CLUSTER = 1		Se Lat.> 44.7
	CLUSTER = 2		Se Lat.< 44.7 e 10.5 < Lon. < 11.5
	CLUSTER = 3		Se Lat.< 44.7 e Lon. > 11.5
===============================================================================
	MAGGIO-LUGLIO

	CLUSTER = 0 		Stazioni di montagna HSTAZ > 500. metri
	CLUSTER = 1		Se Lon. < 10.2 		
	CLUSTER = 2		Se 10.2 < Lon. < 11.2
	CLUSTER = 3		Se Lon. > 11.2
===============================================================================

	dove Lat. e Lon. sono  le coordinate delle stazioni in gradi
	centesimali e HSTAZ le altezze delle stazioni

===============================================================================
	INPUT:

	K		I*4	Numero stazioni da suddividere in cluster
	LAT(K)	     Real*4	Vettore latitudine stazioni espresse in
				gradi centesimali
	LON(K)	     Real*4	Vettore longitudine stazioni espresse in
				gradi centesimali
	HSTAZ(K)     Real*4	Vettore altezze stazioni espresse in metri
	IMESE		I*4	Mese relativo alla suddivisione in cluster

===============================================================================
	OUTPUT:

	ICLUSTER(K)	I*4	Vettore contenente il cluster relativo ad ogni
				stazione
	IER		I*4	IER=0 (non utilizzato)

Torna all'indice

Torna alla voce superiore
