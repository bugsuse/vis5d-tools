cong

Programma per la modifica di alcuni semplici parametri del grib

Il sorgente si trova in: /disk3/patruno/ugrib/cong.f 




 $Date: 2002/10/08 17:29:16 $    $Revision: 1.21 $

 usage:

 cong <nomefilein> <nomefileout> [-piii] [-ejjj] [-akkk] [-blll]
      [-step] [-cfm] [-c2a] [-a2c] [-uv2h] [-destag] [-rnnn] [-v]

 <filein>  file input
 <gribout> file output
 -piii     forcing iii generating process
 -ejjj     forcing jjj emission centre
 -akkk     forcing kkk grid definition
 -blll     forcing lll predetermined bit map
 -step     recalculate step grid from extreme
 -cfm      set component flag to m (0/1)
 -c2a      positioning u,v on H grid when C grid
 -a2c      repositioning of u,v from H grid when C grid
 -uv2h     interpolate U/V point variables on H points
           to use sometimes in conjunction with -c2a
 -destag   destag grid
 -rnnn     rigriglia il campo mediando su nnnXnnn punti
 -comp     comprimi i dati ricalcolando il numero di bit significativi
 -/+"setkey <sintassi comando ged>"  esegue il comando setkey del GED (grib editor)
                        ed elabora solo i grib corrispndenti a tale chiave
                        escludendoli o includendoli
               [data:d1,d2,d3] [ora:o1,o2] [scadenza:s1,s2,s3,s4]
               [livello:l1,l2,l3] [variabile:v1,v2,v3]
  Imposta la chiave per la selezione dei grib.
  Tutti i parametri sono opzionali e le virgole possono essere sostituite
  da spazi.
  Tutti i campi non definiti assumono valore "giolli". I campi a sinistra
  dei due punti possono essere abbreviati fino ad essere rappresentati
  con un solo carattere; i campi a destra dei due punti sono definiti come
  segue:

        d1      giorno          )
        d2      mese            )
        d3      anno                    )  emissione
        o1      ora             )
        o2      minuti          )
        s1      indicator of unit of time range (table 4)
        s2      periodo di tempo 1
        s3      periodo di tempo 2
        s4      time range indicator            (table 5)
        l1      indicator of type of level      (table 3)
        l2      height, pressure etc. of levels
        l3      height, pressure etc. of levels
        v1      identification of originating/generating
                                         centre (table 0)
        v2      table 2 version number
        v3      parameter                       (table 2)

                valore negativo assume significato di "giolli"

        si ppossono inserire commenti facendoli precedere dai
        caratteri speciali # e !

    Vengono eseguiti prima i test di esclusione (-) (se presenti)
    e poi quelli di inclusione (+) (se presenti).


 -f<nome file>  legge le opzione della funzione setkey dal file
           specificato
 -afa      set formatted output AFA standard (ascii)
 -v        set verbose mode
Vedi anche: displgrib ged
